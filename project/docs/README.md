# 销售分析仪表盘

## 项目简介
本项目基于 Python 和 Streamlit 实现了一个交互式销售数据分析与可视化仪表盘，功能包括数据上传、销售分析、客户分析、市场篮子分析、销售预测及 AI 智能洞察辅助。

---

## 功能模块

| 模块 | 功能 |
|------|------|
| Home | KPI 总览、功能入口 Grid 卡片 |
| Dataset Center | 数据上传、内置数据集加载、预览与质量报告 |
| Sales Analysis | 月度销售趋势、国家销售分布、TOP10 产品分析、销售额占比 |
| Customer Analysis | 客户贡献、RFM 分析、购买行为洞察 |
| Market Basket Analysis | TOP 关联组合分析、销售贡献 TreeMap |
| Forecast Analysis | 销售预测模型及趋势分析 |
| AI Assistant | 基于 Qwen 千问 API 的智能问答 |
| About | 项目介绍、技术栈、架构图、功能亮点 |

## 安装与运行

1. 克隆仓库

```bash
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名
````

2. 创建虚拟环境并安装依赖

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate

pip install -r requirements.txt
```

3. 设置 API Key（可选）

* 环境变量方式：

```powershell
# Windows
$env:QIANWEN_API_KEY="你的千问 API Key"
# macOS / Linux
export QIANWEN_API_KEY="你的千问 API Key"
```

* Streamlit Secrets 方式：

在 `.streamlit/secrets.toml` 中添加：

```toml
QIANWEN_API_KEY = "你的千问 API Key"
# 或者
[qwen]
api_key = "你的千问 API Key"
```

> 没有 API Key 时，AI Assistant 会自动使用本地规则分析。

4. 启动应用

```bash
python -X utf8 -m streamlit run project/app.py
```

浏览器访问：`http://localhost:8501`

### AI 的价值与局限反思

在本项目开发过程中，AI 工具（ChatGPT、Codex、 等）在多个环节对我提供了极大帮助，但也存在一定局限。以下是具体总结：

#### 1. AI 提升效率的环节
- **代码生成与模板搭建**：AI 可以快速生成页面框架、API 调用模板、可视化图表代码片段，节省了大量重复性编码时间。  
- **思路启发与方法参考**：AI 提供了分析方法示例、Prompt 设计思路和可复用组件建议，使我能快速确定实现方向。  

总体而言，AI 最帮助我提升效率的是**初步代码实现、功能模板搭建和页面布局建议**，让我能把更多精力集中在逻辑设计和核心分析上。

#### 2. AI 最容易误导的地方
- **生成完整模块时容易出现逻辑错误**：一次性生成整页功能，代码可能运行超时或逻辑与现有系统不兼容。  
- **默认参数或业务逻辑不贴合项目需求**：AI 提供的默认布局、参数配置或指标选择并不总是符合我设计的分析目标，需要手动修改。  
- **忽略已有约束或边界条件**：例如缓存逻辑、环境变量读取、异常处理等细节，AI 生成的代码常常缺失或不完善。  
- **过于复杂的交互或布局方案**：AI 有时倾向生成过度复杂的界面，反而增加修改负担。

#### 3. 如果重新做一遍，我会更高效地使用 AI
- **分块 Prompt**：每次只让 AI 生成单个页面或单个功能模块，避免一次性运行长时间导致错误或难修改。  
- **明确输入与输出边界**：先确定好页面功能、布局、数据输入输出，再生成对应代码，提高生成准确性。  
- **主动筛选与验证**：将 AI 生成的代码视作初稿，必须经过验证、修改和整合，确保符合项目需求。  
- **文档与注释辅助生成**：使用 AI 生成代码时，同时生成清晰注释和文档，方便自己理解和修改。  
- **结合版本控制**：在每次 AI 生成后立即 Commit，以便追踪修改和回退。

总结：AI 是一个强大的助理，能大幅提高编码和设计效率，但在关键逻辑、分析思路和决策环节仍需由我们主导；合理拆分任务、验证结果和精细管理 Prompt 是充分发挥 AI 能力的关键。
